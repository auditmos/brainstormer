# Question Format Reference

## Numbering

Number every question sequentially across the entire session: Q1, Q2, Q3…

## Answer Options

When proposing answer options, number them. Example:

```
**Q3.** What's the data sensitivity level?
  1. Public
  2. Internal only
  3. Contains PII
  4. Regulated (HIPAA, PCI, etc.)
```

## Client Replies

Client can reply with just the number (e.g. "3" or "Q3: 3"). Free-form answers always accepted.

## Multiple Questions

Ask one question at a time. Wait for a response before moving to the next. Exception: tightly coupled yes/no follow-ups can be grouped (still numbered individually).
